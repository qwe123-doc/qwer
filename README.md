## Описание
Это простое GUI-приложение для хранения информации о фильмах. Позволяет добавлять фильмы, фильтровать их по жанру и году выпуска, а также сохранять и загружать данные в JSON-файл.

## Использование
1. Запустите приложение.
2. Введите название, жанр, год выпуска и рейтинг фильма.
3. Нажмите «Добавить фильм».
4. Используйте поля фильтров для просмотра фильмов по жанру или году.
5. Для сохранения данных нажмите «Сохранить».
6. Для загрузки данных из файла — «Загрузить».

## Заметки
- Год должен быть числом.
- Рейтинг — число от 0 до 10.
